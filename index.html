MatlabJS

A Javascript mini-library

Description

MatlabJS is a mini-library with common functions for Matlab® / GNU Octave users who are new to Javascript. This utility is currently at version loading..

Examples:

The library is already loaded in this webpage.
Open the browser console using right-click+inspect.
Try out these examples: 
                
    tic() 
    a=linspace(0,1) 
    disp(a)
    b=eye(8) 
    c=rand(8) 
    d=mul(b,c) 
    toc()
Perform a quick check to see if the current version is stable.

How to use this in your project:

Method 1 - CDN (recommended):

Include the following script tag within your html head. 
     <script src="https://cdn.jsdelivr.net/gh/VivekTRamamoorthy/MatlabJS/Matlab.js"></script>
This would only give the standard functions.
To use plotting tools and linsolve function, additionally include plotlib.js and ndarray.js files. 
     <script src="https://cdn.jsdelivr.net/gh/VivekTRamamoorthy/MatlabJS/ndarray.js"></script>   
     <script src="https://cdn.jsdelivr.net/gh/VivekTRamamoorthy/MatlabJS/plotlib.js"></script>

Method 2 - File inclusion (robust):

Download Matlab.js file from the MatlabJS repository to your project and and include it using a script tag as below. 
                     <script src="Matlab.js"></script>

Node

Install using the following command in your terminal and use const MatlabJS = require("matlabjs") in your modules. 
                  npm install matlabjs

Limitations:

  • Unfortunately, operator overloading is not permitted in JavaScript, and we cannot yet write c=A*b for matrices and arrays. Instead we have to resort to using universal functions c=mul(A,b) that perform actions based on the input type.

  • The code is not optimised for efficiency.

  • This project is in its priliminary stage. More functionalities will be included as time permits. Use it with caution.
