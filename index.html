MatlabJS

Use common Matlab functions in Javascript

Matlab® and GNUOctave are a high-level programming language used by scientists and engineers. It allows efficient scientific computing in very few lines of code.

For Matlab® / GNU Octave users who are new to Javascript, things may not be easy. Javascript was simply designed for a different purpose!

But no worries! The purpose of MatlabJS is to help Matlab users get a jumpstart in Javascript right away with (almost) the same syntax.

Examples:

Try out these examples in the console:
            
                a=linspace(0,1,100) 

                disp(a) 

                b=eye(8); 

                c=rand(8); 

                d=mul(b,c)

How to use in your project

  • Add Matlab.js file from MatlabJS to your project.
  • Throw in a script tag:
    •             
                <script src="Matlab.js"></script>
  • Thank me later.

    • Drawbacks:

  • Unfortunately, operator overloading is not permitted in JS.
    so we cannot yet do c=A*b for matrices and instead have to do c=mul(A,b).
  • The code is not optimised for efficiency.
  • This project is in its primitive stage.
    More functions will be included as time permits.
